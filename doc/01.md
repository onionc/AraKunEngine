## 编译

第五课创建了一些目录和空框架，需要编译。
我是Windows下编译，编译时有两种选择。
1. `cmake ..`（在我电脑上默认使用VS）；build目录下生成了一个VS项目，可以打开编译，也可以`cmake --build .`编译
2. `cmake -G ' cmake -G "Unix Makefiles" ../`, 再使用 `make` 编译。

写了两个简单的脚本

build.bat
```
mkdir build
cd build

cmake -G "Unix Makefiles" ../

make

cmd /k
```

buildVs.bat
```
mkdir buildVs
cd buildVs

cmake ..

cmake --build .

cmd /k
```
